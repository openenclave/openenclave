# OE_SpinDestroy()

Destroy a spin lock.

## Syntax

    xxxintxxx OE_SpinDestroy(OE_Spinlock *spinlock)
## Description 

This function destroys a spin lock.



## Parameters

#### spinlock

Destroy this spin lock.

## Returns

Return zero if successful
